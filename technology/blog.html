  • 如何使用vuePress搭建个人博客

    # vuePress是什么？

    vuePress是vue为驱动的一个静态网站生成工具，它的开发者是vue的作者尤雨溪，我们可以用它来编写技术文档和搭建个人博客。

    # 特性：

    • 内置 markdown 扩展，针对技术文档进行了优化
    • 可以直接在markdown 文件中使用Vue代码
    • 包含一个默认主题，也可通过vue代码编写自定义主题
    • 支持PWA
    • Google Analytics 集成

    # 正式开始

    # 首先要全局安装vuePress,要保证你的Node.js版本>=8

    yarn global add vuepress # 或者：npm install -g vuepress

    # 接下来为我们的博客创建一个本地项目

    创建项目目录

    mkdir myBlog
cd myBlog

    初始化项目

    npm init 或者： yarn init

    创建docs文件夹, docs文件夹相当于项目的根目录

    mkdir docs

    在docs目录下创建README.md文件，用作博客的主页

    echo '# Hello VuePress' > README.md

    在docs目录下创建一个.vuepress文件夹，在该文件夹下创建一个config.js文件用来写配置信息，创建一个public文件用来存放图片等静态资源。

    mkdir .vuepress
cd .vuepress
touch config.js
mkdir public

    现在项目的结构如下所示：

    myBlog
      -----docs
         ----.vuepress
           ----public    // 图片等静态文件
           ----config.js    // 配置文件
         ----README.md    // 主页
      ----package.json

    # 运行项目

    在package.json中添加项目运行和build的脚本

    {
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

    执行命令运行项目就可以开始编写文档了，访问http://localhost:8080即可以看到网页效果

    npm run docs:dev

    执行build命令,生成网站的静态资源，位于.vuepress/dist目录下

    npm run docs:build

    # 添加配置，美化页面

    基本配置，写在.vuepress/config.js里

    module.exports = {
  title: 'Mei Blog',
  description: '刘淑美——前端一枚',
  head: [
    ['link', { rel: 'icon', href: `/logo.jpeg` }],
    ['link', { rel: 'manifest', href: '/manifest.json' }],
    ['meta', { name: 'theme-color', content: '#3eaf7c' }],
    ['meta', { name: 'apple-mobile-web-app-capable', content: 'yes' }],
    ['meta', { name: 'apple-mobile-web-app-status-bar-style', content: 'black' }],
    ['link', { rel: 'apple-touch-icon', href: `/icons/apple-touch-icon-152x152.png` }],
    ['link', { rel: 'mask-icon', href: '/icons/safari-pinned-tab.svg', color: '#3eaf7c' }],
    ['meta', { name: 'msapplication-TileImage', content: '/icons/msapplication-icon-144x144.png' }],
    ['meta', { name: 'msapplication-TileColor', content: '#000000' }]
  ],
  serviceWorker: true,
  themeConfig: {
    nav: [
        { text: '主页', link: '/' },
        { text: '文章',
          items: [
            { text: '前端技术', link: '/technology/' },
            { text: '方法论', link: '/methods/' }
          ] 
        },
        { text: '关于', link: '/about/' },
        { text: 'Github', link: 'https://github.com/liushumei' },
    ],
    sidebar: {
      '/methods/': [['time', '时间管理'], ['xmind', '思维导图']],
      '/technology/': [['blog', '如何使用vuePress搭建个人博客']]
      },
    sidebarDepth: 1,
    lastUpdated: 'Last Updated'
  }
}
    • title: 网站的标题
    • description: 网站的描述
    • head: 被注入页面 HTML额外的head标签, 格式为[tagName, { attrName: attrValue }, innerHTML?]
    • serviceWorker: 如果设置为 true，VuePress 将自动生成并注册一个 service worker ，这个 worker 将内容缓存以供离线使用（仅在生产环境中启用）。
    • themeConfig: 主题设置，nav为导航设置，sidebar为侧边栏设置

    # 默认主题

    默认的主题提供了一个首页（Homepage）的布局，想要使用它，需要在你的根级 README.md 的 YAML front matter 指定 home: true。

    ---
home: true
heroImage: /wanzi.jpg   
actionText: 关于我 →
actionLink: /guide/
footer: MIT Licensed | Copyright © 2018-present Evan You
---

    # 自定义主题

    VuePress 使用单文件组件来构建自定义主题。想要开发一个自定义主题，首先在你的文档根目录新建一个 .vuepress/theme 文件夹，然后再创建一个 Layout.vue 文件
